# Document check_origin_integrity force-push as an explicit CLAUDE.md exception

`check_origin_integrity()` in `scripts/lib-agent.sh` force-pushes to
`origin/main` when a tampering revert is needed. This is a deliberate
emergency exception to the "never push to main directly" rule, analogous
to the housekeeping carve-out already documented in CLAUDE.md.

Currently this exception is only noted in a code comment at the push site
(lib-agent.sh line ~292). It is not in CLAUDE.md or OPERATIONS.md, so
future reviewers may flag it as a policy violation.
