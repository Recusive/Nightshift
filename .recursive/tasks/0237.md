# Replace /tmp/recursive_cost_msg with mktemp in daemon.sh

Three reviewers flagged that `/tmp/recursive_cost_msg` is a predictable temp path. A local attacker could pre-create a symlink at that path, causing the daemon to write cost data to an arbitrary file. Impact is bounded (content is only printed to stdout, never executed), but the fix is trivial.

Replace with:
```bash
COST_MSG_TMP=$(mktemp)
# ... 2>"$COST_MSG_TMP"
cat "$COST_MSG_TMP" 2>/dev/null || true
rm -f "$COST_MSG_TMP"
```
