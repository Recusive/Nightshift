# Module map session labels need a monotonic source

`MODULE_MAP.md` currently derives its `session #NNNN` header from the numbered
handoff files present in the repo. Handoff compaction can remove older numbered
files, so the derived label may eventually become less stable than intended.

## Acceptance Criteria
- [x] Module-map headers derive their session label from a monotonic source that survives handoff compaction
- [x] Compaction does not cause the module-map session label to move backward
- [x] Regression tests cover a repo with compacted handoffs and confirm stable labeling
