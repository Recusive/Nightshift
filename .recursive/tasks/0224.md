# Brain never runs nightshift against Phractal target repo

## Problem

The build agent modifies nightshift source code but never actually runs `nightshift run` against the Phractal target repo to verify the tool works end-to-end. The only time it ran was eval #0016 (session #0108).

The whole point of nightshift is to be run against target repos. The system builds the tool but never uses it.

## Fix

The brain should periodically delegate a build task that runs `nightshift run --agent claude` against Phractal with 1-2 cycles. This tests the full pipeline: clone, profile, plan, build, verify.

Schedule: every 5 sessions, or after any change to nightshift/owl/ or nightshift/raven/ (the cycle and feature modules).

## Acceptance Criteria

- Brain delegates a "run nightshift against Phractal" task periodically
- Results (success/failure, what was built, shift log) feed back into the task queue
- Failures in nightshift runs create targeted fix tasks
