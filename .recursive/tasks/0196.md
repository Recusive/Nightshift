# Add .recursive.json to PROMPT_GUARD_FILES

## Context

Pentest scan (2026-04-07, THEORETICAL): `.recursive.json` is not in
`PROMPT_GUARD_FILES` in `Recursive/engine/lib-agent.sh`. An agent could modify
`eval_frequency`, project metadata, or `verify_command` and push to origin/main.
