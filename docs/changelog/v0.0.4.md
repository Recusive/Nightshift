# v0.0.4 -- Agent Quality

**Released**: 2026-04-03
**Tag**: `v0.0.4`
**Status**: Released

Improvements to agent behavior quality: category balancing, shift-log verification fixes, runner hardening.

## Fixed

- **[verification]** Shift-log verification now accepts agents that commit shift-log updates separately from fix commits. Previously every commit had to include the shift log, which caused codex cycles to be incorrectly rejected. Now requires at least one commit per cycle to include the shift log.
- **[verification]** Shift-log-only commits no longer count toward max_fixes_per_cycle limit.
- **[verification]** expected_cycle_commits returns a (min, max) range to tolerate separate shift-log commits.

## Added

- **[intelligence]** Category balancing directive: steers the agent toward underrepresented fix categories after a configurable cycle threshold (`category_balancing_cycle`, default 3). Prioritizes unexplored categories from CATEGORY_ORDER, naming the highest-priority target and up to two more.
- **[runner]** Worktree validation before cycles start (from PR #13)
- **[runner]** Target repo checkout validation with early failure (from PR #13)
- **[runner]** Forbidden repo-wide command detection from both raw output and structured results (from PR #13)
- **[runner]** One-commit-per-fix cycle accounting (from PR #13)
- **[runner]** High-signal focus path hints for short validation runs (from PR #13)
- **[runner]** npm lockfile generation prevention in lockfile-less repos (from PR #13)

## Changed

- **[prompt]** Updated agent instructions to explain shift-log co-committing with explicit git add example
- **[skill]** SKILL.md updated to prefer co-committing shift log but tolerate separate commits

## Internal

- Merged PR #13 (9 commits) with conflict resolution across 4 files
- __all__ in __init__.py sorted alphabetically (ruff RUF022 compliance)
- Simplified high_signal_focus_paths() by removing redundant fallback (PR #20)
- Broadened _extract_shell_command regex to handle bash/sh/dash wrappers (PR #20)
- Added Python/Go/Rust paths to HIGH_SIGNAL_PATH_CANDIDATES (PR #20)
- Added docstring to expected_cycle_commits() documenting commit contract (PR #20)
