Loop 2 scaffolding begins: repo profiling, feature planning, and the foundation for sub-agent coordination.

## Added

- **[profiler]** Repo profiling module (`nightshift/profiler.py`): detects primary language by file extension census, frameworks by marker files and package.json dependencies, package manager, test runner, instruction files (CLAUDE.md, AGENTS.md, etc.), top-level directory structure, monorepo markers, and total file count. Returns a `RepoProfile` TypedDict consumed by all future Loop 2 components.
- **[types]** `RepoProfile` and `FrameworkInfo` TypedDicts for structured repo analysis.
- **[constants]** Profiling data: `LANGUAGE_EXTENSIONS`, `FRAMEWORK_MARKERS`, `FRAMEWORK_PACKAGES`, `INSTRUCTION_FILE_NAMES`, `MONOREPO_MARKERS`, `PROFILER_SKIP_DIRS`.
