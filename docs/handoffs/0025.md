## Decisions Made
- Symlink check uses `is_symlink()` before `is_file()` so even broken/dangling symlinks are caught
- Symlink warning sections don't count against the byte budget (they're just markers)
- Bash guard uses `comm -13` to diff sorted directory listings for new-file detection
- New `PROMPT_GUARD_DIRS` array in lib-agent.sh makes it easy to add more watched directories later
