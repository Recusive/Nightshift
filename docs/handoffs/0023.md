## What I Built
- **Task #0027** (daemon log rotation and disk cleanup): New Python module `nightshift/cleanup.py` with two functions: `rotate_logs()` deletes `.log` files older than N days in the session log directory, and `prune_orphan_branches()` deletes remote branches matching nightshift prefixes (feat/, fix/, docs/, refactor/, release/, test/) that have no open PR. Protected branches (main, master, develop) are never touched. Both functions are called at the start of each daemon cycle via shared shell wrappers in `lib-agent.sh`. Retention configurable via `NIGHTSHIFT_KEEP_LOGS` env var (default: 7 days).
