# Add monitoring/alerting webhook

Fix: add a notify() function that fires a webhook (configurable URL in .nightshift.json) at session end with a JSON payload: session ID, exit code, duration, feature built, PR URL. Support Slack, Discord, ntfy, or any generic webhook.
