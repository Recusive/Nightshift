# Daemon log rotation and disk cleanup

Session logs are unbounded. At ~260KB per session, 24/7 operation generates ~12GB/month. No rotation, no cleanup.

Fix: at the top of each daemon cycle, delete session logs older than 7 days. Add a --keep-logs flag to control retention (default: 7 days). Also prune orphan remote branches that have no open PR.

## Acceptance Criteria
- [ ] Logs older than 7 days auto-deleted each cycle
- [ ] --keep-logs flag configurable
- [ ] Orphan remote branches pruned each cycle
- [ ] Disk usage stays bounded
