# Centralize the healer retention default

The default "keep 50 healer entries live" value now exists in both
`nightshift.constants.DEFAULT_KEEP_HEALER_ENTRIES` and shell-side daemon/helper
defaults. That is easy to drift over time.

## Acceptance Criteria
- [ ] The live-healer retention default is defined in one authoritative place
- [ ] Shell helpers and daemon env fallbacks derive from that single source instead of duplicating `50`
- [ ] Regression coverage or docs checks fail if the shell/Python defaults diverge again
