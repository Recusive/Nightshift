# Build feature CLI orchestrator for Loop 2

With profiler, planner, decomposer, spawner, and integrator all complete, build the orchestrator that ties them together. This is the `nightshift build "feature description"` command that runs the full Loop 2 pipeline end-to-end.

See docs/vision/02-loop2-feature-builder.md for the full flow.

## Acceptance Criteria
- [ ] `nightshift build "description"` runs the full pipeline: profile -> plan -> decompose -> spawn waves -> integrate -> verify
- [ ] Each wave is spawned then integrated before the next wave starts
- [ ] Final verification runs full test suite + linting after all waves
- [ ] Feature state is tracked (FeatureState type in types.py)
- [ ] `nightshift build --status` shows current build progress
- [ ] `nightshift build --resume` can resume an interrupted build
- [ ] Human confirmation step before spawning agents (can be bypassed with --yes)
- [ ] Tests cover the orchestrator logic (mocked sub-components)
- [ ] Vision tracker updated
