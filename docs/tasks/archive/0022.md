# Build integrator module for Loop 2

With the spawner complete, build the integrator -- the component that merges sub-agent work after each wave, runs tests, and handles failures. This connects the spawner to the overall build pipeline.

See docs/vision/02-loop2-feature-builder.md "Step 5: Integration" for spec.
