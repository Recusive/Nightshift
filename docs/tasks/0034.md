# Interactive daemon setup — ask for config before running

When you run `./scripts/daemon.sh` with no arguments, it should prompt you interactively before starting:

1. **Which agent?**
   ```
   Which agent should run this shift?
     1) claude
     2) codex
   Enter choice:
   ```

2. **How long?**
   ```
   How long should the daemon run?
     1) 2 hours
     2) 4 hours
     3) 6 hours
     4) 8 hours (overnight)
     5) Unlimited (until I stop it)
   Enter choice:
   ```

3. **Confirm and start.**
   ```
   Starting Nightshift daemon:
     Agent:    claude
     Duration: 8 hours
     Sessions: ~16 (estimated at 30 min each)
   
   Press Enter to start or Ctrl+C to cancel.
   ```

If arguments are passed (e.g., `./scripts/daemon.sh claude 60 10`), skip the interactive prompts and use the args directly — backward compatible.

The duration translates to a max-sessions count: duration_hours * 60 / average_session_minutes. Use 30 min as the estimate.

Apply the same interactive setup to `daemon-review.sh` and `daemon-strategist.sh`.

## Acceptance Criteria
- [ ] Running `./scripts/daemon.sh` with no args prompts for agent and duration
- [ ] Running `./scripts/daemon.sh claude 60 10` skips prompts (backward compatible)
- [ ] Duration converts to session count
- [ ] All three daemon scripts have the interactive setup
- [ ] Confirmation prompt before starting
