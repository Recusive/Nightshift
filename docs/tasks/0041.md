# Add cache_read pricing assertions for gpt-5.4-mini and gpt-5.4-nano

The constant tests for gpt-5.4-mini and gpt-5.4-nano only verify `input` and `output` pricing. The `cache_read` values (0.075 for mini, 0.02 for nano) are never asserted. A pricing typo in those fields would go undetected.
