# Add integration test: prompt-guard detects check.sh modification

## Context

PR #135 added `scripts/check.sh` and `scripts/format-stream.py` to
`PROMPT_GUARD_FILES`. The reviewer noted that `save_prompt_snapshots` and
`check_prompt_integrity` both iterate `PROMPT_GUARD_FILES` (lib-agent.sh:55
and :86), so the snapshot+compare path should cover the new files. However,
there is no automated test that confirms end-to-end behavior when an existing
guarded script is modified.

## Acceptance Criteria

- A test (shell or Python) modifies `scripts/check.sh` after
  `save_prompt_snapshots` runs and then calls `check_prompt_integrity`.
- The test verifies that `check_prompt_integrity` returns non-zero and writes
  an alert file containing a diff of the change.
- The test cleans up (restores original, removes temp files).
- `make check` continues to pass.
